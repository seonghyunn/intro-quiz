YouTube API 키 발급 방법 (무료 · 약 5분)
  1. ① 프로젝트 만들기 — 구글 로그인 후 이름은 아무거나 입력하고 "만들기"
  2. ② YouTube Data API v3 열기 — 파란 "사용(Enable)" 버튼 클릭
  3. ③ 사용자 인증 정보 열기 — "+ 사용자 인증 정보 만들기" → "API 키" → 생성된 키 복사
  4. 이어지는 "API 키 제한" 화면에서 API 제한사항 → 키 제한을 선택하고 목록에서 YouTube Data API v3를 체크한 뒤 저장 (키를 이 용도로만 쓰도록 잠그는 안전장치예요)
  5. 복사한 키를 아래 칸에 붙여넣고 저장

무료이며 하루 약 100곡 검색 한도가 있어요. 키는 이 브라우저에만 저장되고 어디에도 전송·공개되지 않아요.
